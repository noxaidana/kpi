Принцип мотивационной программы

  1. Выставляется общий план на отдел продаж.
  2. Учёт плана — методом «по начислению»: каждое начисление фиксируется за менеджером, который ведёт сделку, по которой пришло начисление.
  3. В зависимости от выполнения плана на отдел начисляется сумма бонуса согласно грейду.
  4. Относительно начисленной выручки вычисляется процент доли каждого менеджера.
  5. Общая сумма бонуса распределяется по менеджерам в том же процентном соотношении, в котором вычислена выручка каждого.

💾 Экспорт данных для сайта

  1. Нажмите «Скопировать».
  2. Откройте index.html на GitHub → карандаш (Edit).
  3. Найдите блок между /*=НАЧАЛО ДАННЫХ=*/ и /*=КОНЕЦ ДАННЫХ=*/ и замените его вставленным.
  4. Commit changes. Через ~1 минуту сайт обновится для всех.
